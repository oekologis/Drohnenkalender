Einrichtung (einmalig)

Ihr braucht ein kostenloses Supabase-Konto für die geteilte Datenbank. Folgt diesen Schritten:

1
Geht auf supabase.com → New Project anlegen (kostenlos)
2
Im Projekt: SQL Editor öffnen und diesen Code ausführen:
create table bookings ( id uuid default gen_random_uuid() primary key, date text not null, drone text not null, label text not null, created_at timestamptz default now() ); alter table bookings enable row level security; create policy "public read" on bookings for select using (true); create policy "public insert" on bookings for insert with check (true); create policy "public delete" on bookings for delete using (true);
3
Geht zu Project Settings → API und kopiert die Werte unten:
